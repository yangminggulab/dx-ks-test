# KuaiRec AB Test 项目说明

用途说明：本文件用于统一记录项目背景、数据来源、关键指标与分析目标，确保每次分析时遵循相同的业务口径。

**每次开始分析前，请先阅读本文件。**

## 1. 项目背景

本项目模拟短视频平台推荐策略的 AB Test 场景。平台会对不同用户随机分配不同推荐策略，并观察实验组与对照组在观看行为、互动行为和留存相关指标上的差异，从而判断新策略是否优于旧策略。

## 2. 数据来源

本项目的数据来源为 **KuaiRec** 数据集。KuaiRec 是面向短视频推荐研究的公开数据集，可用于模拟推荐曝光、播放、互动等业务场景，适合用于推荐系统策略评估和数据分析练习。

## 3. 北极星指标

本项目的北极星指标为 **完播率**。

- 完播率定义：用户完整观看视频的次数 / 视频播放次数。
- 业务意义：完播率能够较好反映推荐内容与用户兴趣的匹配程度，是短视频推荐策略效果评估中的核心指标之一。

## 4. 分析目标

本项目的核心分析目标包括以下两类：

1. **策略评估**：比较实验组与对照组在完播率、播放时长、点击率等关键指标上的表现，判断新推荐策略是否带来显著提升。
2. **用户分层**：根据用户活跃度、偏好特征、历史观看行为等维度进行分层，分析新策略在不同用户群体中的效果差异，识别受益人群和潜在风险人群。

## 5. 分析提醒

- 所有实验分析应先确认分流是否随机且组间样本量是否平衡。
- 正式开展 AB Test 结论分析前，建议先进行 A/A Test 以检查实验系统稳定性。
- 输出结论时，需要同时关注统计显著性与业务显著性，避免只看 p-value 而忽略实际收益大小。
